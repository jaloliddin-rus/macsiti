Guide to add "Back to Top" Anchor

1. link css and javascript files to your web page

CSS: <link href="https://seamonkeyhd.github.io/macsiti/back2top.css" rel="stylesheet" type="text/css"/>

Javascript: <script src="https://seamonkeyhd.github.io/macsiti/back2top.js"></script>


2. add this line of code after your content 

<button onclick="topFunction()" id="myBtn" title="Back to Top"><img src="https://seamonkeyhd.github.io/macsiti/arrow.png"/></button>

3. save and test
